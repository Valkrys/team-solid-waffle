/roles

Collection of jobs and all the capabilities and families they belong to

/roles get

get

Gets an array of JSON objects containing roleID, roleName, capabilityID, capabilityName, jobFamilyID and jobFamilyName from the DB

get /roles

Gets an array of JSON objects containing roleID, roleName, capabilityID, capabilityName, jobFamilyID and jobFamilyName from the DB

  • Response

HTTP status code 200

Body

Media type: application/json

Example:

[
    {
        "roleID": 1,
        "roleName": "Software Engineer", 
        "jobFamilyID": 2, 
        "jobFamilyName": "Technical",
        "capabilityID": 1, 
        "capabilityName": "Apprentice"
    },
    {
        "roleID": 2,
        "roleName": "Software Engineer", 
        "jobFamilyID": 2, 
        "jobFamilyName": "Technical",
        "capabilityID": 2, 
        "capabilityName": "Trainee"
    }
    
]
